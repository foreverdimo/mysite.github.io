## 第一次作业

选择一个长篇文本，作为分析对象进行词频统计，并进行分析。

因为我对宇宙物理学有着浓厚的兴趣，我选择的文本为《时间简史》。程序对文章进行读取，读取后用jieba库的lcut函数进行分词，再去除一些诸如代词副词等非关键词汇后利用字典对词频进行统计，并按照词频排序后存在列表中，此时可以根据用户的需要输出出现次数最多的词汇，并存在csv文件中。
